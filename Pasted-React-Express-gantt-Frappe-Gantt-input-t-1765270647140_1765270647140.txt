React + Express の生産管理アプリです。
/gantt 画面で Frappe Gantt を使って案件別ガントチャートを表示しています。

前回までの修正で、
- 開始日（input type="date"）
- 終了日（input type="date"）
- 案件名で絞り込み（input type="text"）
を含んだツールバーを追加してもらいました。

しかし、ブラウザで確認すると、

- 画面を開いた「最初の 1 秒くらい」だけ、
  開始日・終了日・案件名の入力フィールドが表示される
- その後すぐに消えてしまい、
  結局 [←] [Today] [すべて] だけのシンプルなツールバーだけが残る

という状態になっています。

これは、
- 初期レンダリング時には「新しいツールバーコンポーネント」が一度表示されるが、
- その後、別のコンポーネント（古いツールバー）が再レンダリングされて上書きされている

という挙動だと思います。

【お願いしたいこと】

/gantt 画面に関わるコンポーネント（ページコンポーネント、レイアウト、ヘッダー周り）を確認し、

1. 「ガントチャートのタイトルとツールバー」を描画している部分を一箇所に統合してください。
   - 具体的には、
     - 開始日 / 終了日 / 案件名入力を含む“新しいツールバー”を
       ガントチャートの正式なツールバーとして採用する
     - それ以外の、古いツールバー（[←] [Today] だけのものなど）は削除するか、
       新しいツールバーの中に取り込むようにしてください。

2. ページの初期ロード時・データ取得後などに再レンダリングされても、
   「開始日・終了日・案件名入力を含むツールバー」が常に表示され続けるようにしてください。
   - 条件付きレンダリング（if (loading) ... など）で旧ツールバーに切り替わっていないか確認し、
     その条件分岐を整理してください。

3. JSX 構造としては、概ね次のようなイメージに収束させてください（実際の構造はプロジェクトに合わせてOKです）：

   <PageLayout>
     <header className="page-header">
       <h1>案件別ガントチャート</h1>
       <p>frappe-ganttで案件別タイムラインを表示</p>

       <div className="gantt-toolbar">
         <div className="gantt-toolbar-left">
           <button>←</button>
           <button>Today</button>
           <button>すべて</button>
         </div>
         <div className="gantt-toolbar-right">
           <label>
             開始日
             <input type="date" ... />
           </label>
           <label>
             終了日
             <input type="date" ... />
           </label>
           <input type="text" placeholder="案件名で絞り込み" ... />
         </div>
       </div>
     </header>

     <div className="gantt-wrapper">
       <FrappeGantt tasks={visibleTasks} ... />
     </div>
   </PageLayout>

   重要なのは、
   - /gantt ページ上でツールバーを描画しているコンポーネントが「1箇所だけ」になること
   - そこに必ず開始日・終了日・案件名入力を含めること
   です。

4. そのうえで、
   startDate / endDate / projectFilter の state と、
   visibleTasks のフィルタリング処理が、
   実際に FrappeGantt に渡す tasks に反映されていることも再度確認してください。
   （フィルタのロジック自体は前回指示した内容で構いません）

最終的に、
- 画面のロード直後からずっと、
  開始日・終了日・案件名入力がヘッダー下のツールバーに表示され続ける
- 日付や案件名を入力すると、ガントチャートに表示される案件が絞り込まれる

という状態になるように /gantt 関連コンポーネントとレイアウトを修正してください。
